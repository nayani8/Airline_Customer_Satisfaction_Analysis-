# ✈️ Airline Passenger Satisfaction Analysis

## 📖 Project Overview

This project focuses on analyzing airline passenger satisfaction data to understand the factors that influence customer experience and satisfaction levels.

The analysis explores passenger demographics, travel characteristics, service ratings, delays, and overall satisfaction to identify key insights that can help airlines improve customer experience and service quality.

---

## 🎯 Objective

The main objective of this project is to:

* Analyze passenger satisfaction trends.
* Identify factors affecting customer satisfaction.
* Evaluate airline service ratings.
* Understand the impact of delays on passenger experience.
* Generate actionable business insights through data analysis and visualization.

---

## 📊 Dataset Information

The dataset contains passenger information such as:

* Gender

* Type of Travel
* Travel Class
* Flight Distance
* Departure Delay
* Arrival Delay
* Service Ratings
* Passenger Satisfaction

### Target Variable

* Satisfaction

## 📈 Key Insights

* Online Boarding showed a strong positive relationship with passenger satisfaction.
* Inflight Entertainment and Seat Comfort significantly influenced customer experience.
* Departure and Arrival Delays negatively impacted satisfaction levels.
* Business-class passengers generally reported higher satisfaction ratings.
* Service quality plays a crucial role in improving overall passenger satisfaction.
